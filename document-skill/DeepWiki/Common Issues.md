# Common Issues

## ​ I’m unable to connect my GitHub.com organization

- Navigate to the Devin Enterprise or Organization with the active integration

- Navigate to the Integrations page If you are on an Enterprise account, navigate to https://app.devin.ai/settings/connected-accounts in Enterprise Settings If you are on a Teams account, navigate to https://app.devin.ai/settings/integrations in Organization Settings

- If you are on an Enterprise account, navigate to https://app.devin.ai/settings/connected-accounts in Enterprise Settings

- If you are on a Teams account, navigate to https://app.devin.ai/settings/integrations in Organization Settings

- Click through on the GitHub integration card

- Click “Disconnect” under the GitHub integration

- Go to the GitHub Integration settings

- Navigate to Devin.ai Integration and click “Configure”

- Scroll to the “Danger zone” section to uninstall the integration

- If you are on an Enterprise account, navigate to https://app.devin.ai/settings/connected-accounts in Enterprise Settings

- If you are on a Teams account, navigate to https://app.devin.ai/settings/integrations in Organization Settings

- Click through on the Slack integration card
