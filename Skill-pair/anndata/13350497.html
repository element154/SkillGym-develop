Description

Collection of h5ad files for RNA and protein single-cell perturbation datasets on scPerturb.org.

For the associated publication, see https://www.nature.com/articles/s41592-023-02144-y.

H5ad files were created using scanpy 1.9.1, using gzip compression.

 

Notes

Version notes: 1.4 adds more RNA datasets, corrects errors in SrivatsanTrapnell2020_sciplex3 and ShifrutMarson2018
