Description

These are the count matrices (in .h5ad format) used to produce the results and figures in the paper entitled "distinct mesenchymal cell states mediate prostate cancer progression". 

  • adata_mouse.h5ad: the normalized count matrix and cell annotations for the mesenchymal prostate data from all mouse models.
  • adata_mouse_myo.h5ad: the normalized count matrix for the myofibroblasts and pericytes (both are c0).
  • scenic_mesenchyme_auc.csv: the auc matrix of the regulons activity in the different cells and clusters in the mouse mesenchymal data.
  • scenic_mesenchyme_binary.csv: the binary matrix of regulons activity in the mouse mesenchymal data.
  • adata_human.h5ad: the normalized count matrix and cell annotations for the mesenchymal human primary prostate cancer data.
  • adata_human_boneMet_stroma.h5ad: the normalized count matrix and cell annotation for the bone metastasis data derived from Kfoury et al.
  • visium_PRN.h5ad: the normalized count matrix and cell annotation for the Visium spatial transcriptomics data of the PRN mouse model.
  • visium_WT.h5ad: the normalized count matrix and cell annotation for the Visium spatial transcriptomics data of the WT (PRN-WT) mouse model.
  • natHistory_expression: the normalized expression matrix for the natural history cohort.
  • natHistory_label: the metastasis indicators for the natural history cohort.
