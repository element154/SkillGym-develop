Files too large?

Restrict data to a specific organism:
You can restrict the downloaded data to contain only the information related to your organism of interest. The filtered files will be prefixed with the taxonomic identifier of the selected species. Choose an organism here:

 

    
 
+ Need more options? Click here.
Filter interactions based on the combined confidence score:
While all links in STRING are meaningful, some analyses might require only higher-confidence interactions, excluding the more exploratory part of the interaction spectrum. This can reduce file sizes by up to 90%. Note: Confidence scores in the files below are multiplied by 1000.
Remove redundant interaction pairs (AB-BA):
By default, the links files encode interactions in both directions (AB and BA). However, since functional and physical interactions are symmetric, recording them once (AB) is sufficient without losing information. This reduces links file sizes by 50%.
Choose column delimiter:
Different analysis tools may require specific delimiters (column separators). Select your preferred delimiter type for the links files to ensure compatibility with your analysis tools.   Note: This choice does not impact the content or file size.
