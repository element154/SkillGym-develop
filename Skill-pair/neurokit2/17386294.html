Description

Merging Refactored Code: A Public EEG Microstate Dataset for Studying Developers’ Brain Network Dynamics
 

🧠 1. Overview of the Study

The study “Merging Refactored Code: A Public EEG Microstate Dataset for Studying Developers’ Brain Network Dynamics” investigates the brain network dynamics of software developers while performing code refactoring and merging tasks.
The goal is to identify neural patterns and cognitive load variations associated with decision-making and code comprehension processes during these development activities.

The eeg_experiment.zip dataset contains raw EEG recordings collected from participants, and the 1_ProcessingFiltersEEG.ipynb notebook implements the filtering and preprocessing pipeline required to reproduce and prepare the signals for microstate analysis.

📂 2. Dataset Structure (eeg_experiment.zip)

The dataset is organized as follows:

eeg_experiment/
│
├── EEG/
│   ├── S01/
│   │   ├── S01-1-2024.csv
│   │   ├── S01-2-2024.csv
│   │   └── ...
│   ├── S02/
│   ├── ...
│   └── S42/
│

  • Each folder SXX/ corresponds to a participant.

  • Each .csv file represents one experimental session, containing multiple EEG channels and temporal samples.

3. EEG Processing Pipeline (1_ProcessingFiltersEEG.ipynb)

The notebook 1_ProcessingFiltersEEG.ipynb provides a reproducible implementation of the EEG preprocessing steps using Python, MNE, NumPy, Pandas, and Pycrostates libraries.

Abstract

Understanding how developers manage complex software integration tasks requires insights into the neural mechanisms involved in software development. Although cognitive neuroscience and neuro-software engineering have advanced, there remains a lack of publicly accessible neural datasets capturing brain activity during ecologically complex tasks, such as merging refactored source code. This limitation affects reproducibility and hinders the development of cognitive-aware tools for high-cognitive-load scenarios. We introduce BrainMergeEEG, the first open EEG microstate dataset collected during refactored code-merging tasks, a highly demanding activity in collaborative software development. The dataset includes recordings from 42 participants who completed resting-state sessions and controlled merge tasks with parallel code modifications, comprising approximately 8.0~GB of EEG data. Data are organized by task condition and behavioral performance, enabling detailed analyses of EEG microstate features, including state duration, global field power, and transition probabilities. To promote transparency and reuse, we provide preprocessing scripts and a fully reproducible experimental pipeline. Preliminary analyses reveal consistent associations between EEG microstate patterns and the merge task performance. BrainMergeEEG establishes a neurocognitive benchmark for studying software integration behavior and supports future research on brain-aware software engineering.
